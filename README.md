# React boilerplate

## Apricot (October 28)

- Redux
- Routes
- SSR

# Requirements

[NodeJS](https://nodejs.org) must be installed.

# Installation

```javascript
git clone https://github.com/florianmaxim/react-jungle -b
cd react-jungle
npm install
```
# Usage

```javascript
npm run build
npm run dev
npm run start
```

# Configuration
```javascript
config.json
```
